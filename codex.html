Codex Setup

Set up Codex

Codex is OpenAI's coding agent for terminal, IDE, desktop, and cloud workflows. Use this path if you have access through a ChatGPT plan or an OpenAI API account. After setup, come back to the healthcare workshop home page for the prompt examples and project ideas.

Step 1

Install Node.js

The npm installation path for Codex uses Node.js. Go to nodejs.org and download the LTS version for your operating system. Run the installer and accept all defaults.

To verify it installed, open your terminal and run:

node --version

If you see a version number, you are ready for the next step.

Step 2

Install Codex

In Terminal, Windows Terminal, or PowerShell, run:

npm install -g @openai/codex

When it finishes, you should see a success message with no red errors.

Step 3

Authenticate

Codex supports two OpenAI sign-in paths. For most students, run codex login and sign in with ChatGPT in the browser. API keys are still useful for API-billed or programmatic workflows.

ChatGPT sign-in

Recommended for students with ChatGPT plan access:

codex login codex

API key sign-in

Use this when billing through an OpenAI Platform account. On macOS or Linux:

export OPENAI_API_KEY=your-key-here codex

On Windows PowerShell:

$env:OPENAI_API_KEY="your-key-here" codex
Your sign-in method affects billing, feature availability, workspace controls, and data-handling policies. For this healthcare workshop, keep projects in workflow support, summarization, monitoring, coding support, and triage routing rather than unrestricted diagnosis or treatment agents.
Step 4 (Optional)

Review sandboxing settings

Codex uses sandbox and approval settings to control what it can do on its own. Common modes are read-only, workspace-write, and danger-full-access.

Inside Codex, type /permissions to inspect or change the mode during a session.

Official docs: OpenAI Codex sandboxing guide

Step 5 (Optional)

Turn off “Improve the model for everyone”

If you are using a personal ChatGPT account and do not want your ChatGPT or Codex activity used to improve OpenAI's models, review your data controls. If you authenticate with an API key, review the OpenAI Platform organization's data settings instead.

  1. Open ChatGPT Data Controls.
  2. Find “Improve the model for everyone” and turn it off.

Official docs: Data Controls FAQ

Step 6

Navigate to a folder and start working

Codex works on the files in whichever folder you are in when you launch it. Navigate there first, then start Codex.

Mac

  1. In Terminal, go to a folder you want to work in:
    cd Desktop
  2. Authenticate and launch Codex as shown above.
  3. Type a prompt and press Enter.
  4. To exit, press Ctrl + C.

Windows

  1. In PowerShell or Windows Terminal, go to a folder you want to work in:
    cd Desktop
  2. Authenticate and launch Codex as shown above.
  3. Type a prompt and press Enter.
  4. To exit, press Ctrl + C.
A simple first test prompt: “List the files in this folder and give me one sentence about what each one might be.”
