Install

Basic setup steps

What students do

  1. Install Claude Code using the official native installer, Homebrew, WinGet, or npm instructions.
  2. Confirm terminal access on their machine.
  3. Run claude --version or claude doctor to verify the install.
  4. Create a clean project folder for the assignment.
  5. Start with a small repo or starter files from class, using synthetic data only.

What students should build first

  1. A narrow healthcare workflow agent.
  2. A structured output format.
  3. At least three failure-case tests.
  4. A written note describing where human review is required.
Recommended first projects: patient message triage, visit summarization, medication reconciliation, or remote monitoring review.
Claude Code requires a paid Claude or Console-supported account. The free Claude.ai plan does not include Claude Code access.

Prompting

Suggested first command

Build a small healthcare workflow agent for patient inbox triage. Use only local files in this repo. Produce structured JSON outputs. Never invent symptoms, diagnoses, or medications. Add tests for urgent red flags and missing data.
